// npm modules used in this set of demos
npm init
npm install express
npm install socket.io
npm install fs
npm install uniqid

// lauch a demo by doing the following:
(1) open a terminal connection
(2) issue the following command:

node application_file.js

... for example:

node 01_click_to_draw_rectangle.js

(3) open your browser to the URL displayed in the terminal (e.g. localhost:3001)
(4) open a second tab to the same URL
(5) test the demo by following the instructions on the HTML page
